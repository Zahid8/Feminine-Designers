# Operations

## Daily Counter Workflow

1. Open `/dashboard`.
2. Review today's deliveries, upcoming deliveries, overdue work, and balances.
3. Use **New Order** for new or returning customers.

## Create an Order

1. Go to `/orders/new`.
2. Search by phone or name.
3. Reuse latest measurements if the customer is returning.
4. Add garment items, rates, discounts, delivery date, and stitching instructions.
5. Enter measurements in the compact grid.
6. Record advance payment and payment mode.
7. Save and print the customer copy.

## Find an Old Order

Use `/orders` and search by receipt number, customer name, or phone number. Open the detail page for payment, status, and receipt actions.

## Reprint a Bill

Open the order detail page and choose Customer Copy, Store Copy, or Combined Print. The print/PDF output is regenerated from saved order data.

## Update Measurements

Use the latest customer measurement section as a starting point. New orders must copy measurements into an order snapshot before saving, so future edits do not alter old receipts.

## Mark Delivered

Open the order detail page and use the status controls. Staff should follow Draft -> New -> In Stitching -> Ready -> Delivered. Delivered orders require admin-level caution to reopen.

## Record Payment

Open the order detail page, add payment amount/mode/reference, and verify the balance due. Overpayment is represented as credit.
